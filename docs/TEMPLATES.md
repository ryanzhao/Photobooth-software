# Template System

Templates are JSON-shaped definitions that describe:
- paper size
- export resolution
- background color/image
- photo slots
- text blocks
- asset blocks (logos, overlays, stickers)
- QR blocks
- bleed

Sample template JSON files live under `docs/templates/`.

Implemented defaults:
- `4x6-single.json`
- `4x6-grid.json`
- `2x6-strip.json`
- `square-collage.json`
- `freeform-demo.json`
